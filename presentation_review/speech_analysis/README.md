# speech_analysis

STT 결과를 가지고 발표 정량 지표를 계산하는 폴더입니다.

## 파일

- `metrics.py`
  - 평균 WPM 계산
  - 구간별 발화 속도 계산
  - 추임새 집계
  - 화자별 발화량/WPM/추임새 정리
  - 슬라이드 또는 문장 구간별 분석용 row 생성

## 기준

발화 속도는 음성 전체 길이를 억지로 나누는 방식이 아니라, STT에서 나온 문장/단어 구간과 실제 시간 정보를 기준으로 계산합니다.  
추임새는 전사문에 남아 있는 표현을 기준으로 세고, 이후 LLM 평가에서 문맥 설명을 붙입니다.
