# materials

발표자료 파일을 읽고, 실제 발표 음성과 얼마나 맞는지 비교하는 폴더입니다.

## 파일

- `extractor.py`
  - PDF/PPT 계열 파일에서 텍스트를 추출합니다.
  - 페이지 또는 슬라이드 단위로 비교할 수 있게 내용을 정리합니다.
- `matcher.py`
  - 발표자료 텍스트와 STT 전사문을 비교합니다.
  - 자료에는 있는데 발표에서 약한 키워드, 발표에는 있는데 자료에 약한 키워드를 계산합니다.

## 결과 활용

자료 일치율, 누락 키워드, 페이지별 반영률은 분석 결과 화면과 PDF 리포트에 같이 들어갑니다.
